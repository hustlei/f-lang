# 变量

## 常量、变量定义

~~~
//常量，编译期求值，运行期间不能改变
const PI=3.14159
const a:Int = 25

//变量
var b:Str = "abc"
var s = "xxxxx"
var x = [1, 2, 3]

//单赋值变量，初始化后值将不能改变
let x = 5
let line = read()
~~~

连续赋值

~~~
const a,b,c = 1,2,3

var x, y = 1.0, "str" 
var i:Int, s:Str = 1, "1"
~~~

## 动态类型

为了在某些特殊情况下更加灵活，支持动态类型。动态创建对象，不检查类型。

~~~
dynamic obj = getApp()
dynamic child obj.getChild()
var dynName = child.name()
~~~

> 一般情况下不建议使用dynamic类型
>
> 在调用activex对象，类似office，autocad等应用，后绑定运行对象的方法时，采用dynamic类型会很方便。
